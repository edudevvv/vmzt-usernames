## 🔥 Features
- HTTP / HTTPS Proxy Support 
- Easy & Fast To Setup

## ✍️ Usage
1. Open `config.json` and edit with your custom settings.
2. Open `proxies.txt` and set your proxies (REQUIRED).
3. Open `main.py` and enjoy.
